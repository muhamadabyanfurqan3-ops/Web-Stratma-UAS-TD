# Simulasi Tower Defense

Aplikasi ini menggunakan framework **Streamlit**.

## Cara Menjalankan
1. Aktifkan virtual environment Anda (`.venv\Scripts\activate`)
2. Jalankan `setup-backend.cmd` untuk install library
3. Jalankan `run-backend.cmd` untuk membuka aplikasi

*Catatan: Karena menggunakan Streamlit, frontend dan backend digabung menjadi satu di dalam folder `backend/app.py`.*
